# 데이터 출처 및 이용 조건

| 데이터 | 제공기관 | 용도 | 저장 위치 |
|---|---|---|---|
| TourAPI 4.0 지역 기반 관광정보 | 한국관광공사 | 축제, 관광지, 문화시설, 쇼핑, 숙박, 레포츠, 여행코스 조회 | `backend/data/*.json` |

원본별 상세 수집 정보와 이용 조건은 데이터 폴더의 `SOURCE.md`를 우선한다. API 응답의 이미지 URL은 원본 데이터의 `firstimage`·`firstimage2`를 그대로 전달한다. 이미지가 없는 경우 `null`로 응답한다.

서울 데이터 검증 결과:

- JSON 파일: 7개
- 전체 콘텐츠: 6,518건
- 축제공연행사: 201건
- 좌표 누락: 0건
- 행사 시작·종료일 제공: 0건

행사 날짜가 원본에 없으므로 `event_start_date`, `event_end_date`, `period`는 `null`이다. `createdtime`이나 `modifiedtime`을 행사 날짜로 대체하지 않는다.
