## 문서와 실제 파일의 차이

제공된 `SOURCE.md`에는 `서울_음식점.json` 1,632건과 총 8,150건이 기재되어 있으나, 실제 ZIP에는 음식점 JSON이 포함되어 있지 않다. 구현과 SQLite 초기 데이터는 실제 존재하는 7개 파일 6,518건을 기준으로 한다.

## 준수 사항

- 화면 푸터와 문서에 한국관광공사 출처 표시
- 원본 JSON 파일의 값 수정 금지
- DB 적재 과정은 필드 정규화이며 원본 파일은 별도 보관
- 거리, 예상 도보시간, 권장 체류시간은 서비스가 계산한 파생정보로 구분
- 외부 지도·날씨 데이터는 사용하지 않음
