# 틈서울 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 서울 공연 전후 30~180분에 방문 가능한 실제 장소 1~2곳을 추천하고 익명 후기를 공유하는 Vue.js 3·FastAPI 서비스를 구현·배포한다.

**Architecture:** Vue.js 3 SPA가 FastAPI REST API를 호출한다. FastAPI는 서울 JSON을 SQLite에 적재하고 공연 좌표와 후보 장소 좌표의 Haversine 거리를 계산해 시간 안에 들어오는 1~2곳을 결정한다. 핵심 추천은 외부 API와 OpenAI 없이 동작하며, OpenAI는 DB 검색 결과를 설명하는 챗봇에만 사용한다.

**Tech Stack:** Vue.js 3, Vite, Vue Router, Axios, FastAPI, SQLAlchemy, Pydantic, SQLite, OpenAI API, Pytest, Vitest, Netlify, Render

```python
    inserted = 0
    for path in directory.glob("서울_*.json"):
        payload = json.loads(path.read_text(encoding="utf-8"))
        for item in payload["items"]:
            if not item.get("mapx") or not item.get("mapy"):
                continue
            session.merge(Location(
                content_id=item["contentid"],
                name=item["title"],
                address=item.get("addr1", ""),
                longitude=float(item["mapx"]),
                latitude=float(item["mapy"]),

                image_url=item.get("firstimage", ""),
                district=item.get("lDongSignguCd", ""),
                source="한국관광공사 TourAPI 4.0",
            ))
            inserted += 1
    session.commit()
    return inserted
```

```python
from math import asin, ceil, cos, radians, sin, sqrt

def haversine_km(a_lat, a_lon, b_lat, b_lon):
    dlat = radians(b_lat - a_lat)
    dlon = radians(b_lon - a_lon)
    lat1, lat2 = radians(a_lat), radians(b_lat)
    value = sin(dlat / 2) ** 2 + cos(lat1) * cos(lat2) * sin(dlon / 2) ** 2
    return 6371 * 2 * asin(sqrt(value))

def estimated_walk_minutes(distance_km, route_factor=1.25, speed_kmh=4.5):
    return ceil(distance_km * route_factor / speed_kmh * 60)
```

```python
    outward = estimated_walk_minutes(distance_between(event, place))
    stay = STAY_MINUTES[place.category]
    if phase == "before":
        return outward + stay + outward
```

```python
def two_place_minutes(phase, event, first, second):
    first_walk = estimated_walk_minutes(distance_between(event, first))
    second_walk = estimated_walk_minutes(distance_between(first, second))
    stay = STAY_MINUTES[first.category] + STAY_MINUTES[second.category]
    if phase == "before":
        return first_walk + second_walk + estimated_walk_minutes(distance_between(second, event)) + stay
    return first_walk + second_walk + stay
```

```javascript
export function buildShareUrl(course) {
  const query = new URLSearchParams({
    performanceId: String(course.performance.id),
    phase: course.phase,
    minutes: String(course.available_minutes),
    category: course.preferred_category,
    placeIds: course.places.map(place => place.id).join(','),
  })

  return `${window.location.origin}/course/shared?${query.toString()}`
}
```

- [ ] **Step 3: Web Share와 복사 fallback 구현**

```javascript
export async function shareCourse(course) {
  const url = buildShareUrl(course)
  const data = {
    title: '틈서울 초단기 코스',
    text: `${course.performance.name} 전후 코스를 확인해보세요.`,
    url,
  }

  if (navigator.share) {
    await navigator.share(data)
    return 'shared'
  }

  await navigator.clipboard.writeText(url)
  return 'copied'
}
```

- [ ] **Step 6: 북마크 구현**

```javascript
import { ref } from 'vue'

const KEY = 'teumSeoulBookmarks'
const initial = JSON.parse(localStorage.getItem(KEY) ?? '[]')
const ids = ref(initial)

export function useBookmarks() {
  const persist = () => {
    localStorage.setItem(KEY, JSON.stringify(ids.value))
  }

  const toggle = postId => {
    ids.value = ids.value.includes(postId)
      ? ids.value.filter(id => id !== postId)
      : [...ids.value, postId]
    persist()
  }

  const isBookmarked = postId => ids.value.includes(postId)

  return { ids, toggle, isBookmarked }
}
```
